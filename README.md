# Commands I forget
1. Run shell command
```
:! <you shell command>
```
2. Enter Terminal
```
:terminal
```
3. Go back to normal mode inside Terminal
```
<C-\><C-n>
```
4. Help with Terminal
```
:help terminal-emulator
```
5. Adding files
```
:Explore
```
- j,k move up and down
- Enter to enter a sub-directory
- - to go up a directory
- d to create a directory
- % to create a file
